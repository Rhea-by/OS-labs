## 观察Linux的行为

编写一个默认版本的程序通过检查内核状态报告Linux内核行为。程序应该在stdout上打印以下值：
1，CPU类型和型号；
2，所使用的Linux内核版本；
3，从系统最后一次启动以来已经经历了多长时间（天，小时和分钟）；
4，总共有多少CPU时间执行在用户态，系统态，空闲态；
5，配置内存数量；当前可用内存数，磁盘读写请求数；
6，内核上下文转换数；
7，系统启动到目前创建了多少进程。

## 命令

1. 直接输出
`./a.out`
或者
`./a.out -s`
2. 自定义interval和duration
`./a.out -l 3 70`

其中3为自定义的interval，70为自定义的duration
